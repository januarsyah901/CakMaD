# Version 1.2.0 - 2026-04-01

## Added
- Multiple document tabs - open several documents simultaneously
- File Explorer sidebar - browse and manage all documents
- Drag & drop file import - open .md files from your computer
- Save as .md file - download documents as markdown files
- Recent files section - quick access to recently edited files
- Rename & delete documents - organize your files
- Tab management - close individual tabs without deleting documents
- Document persistence - all files saved to localStorage automatically

## Fixed
- Improved file handling stability
- Better error handling for file imports

## Changed
- Upgraded from single-document to multi-document application
- Enhanced UI with sidebar and tab system
- Improved document management workflow

## Removed
- (No features removed in this release)

## Notes
- All file storage is client-side (localStorage) - no backend required
- Documents persist across browser sessions
- Recently edited files appear first in sidebar
- Maximum 10 recent files tracked
- Each document gets unique ID for reliable state management
- Full markdown file import/export support
