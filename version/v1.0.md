# Release Notes - v1.0.0

**Date**: 1 April 2026
**Status**: Initial Release

## 🚀 Features

### Core Editing
- **Dual Pane Layout**: Split view with real-time editing on the left and live preview on the right.
- **CodeMirror Integration**: Robust text editor using `@uiw/react-codemirror` with Markdown syntax highlighting.
- **Auto-Save**: Automatic draft saving to `localStorage` to prevent data loss.
- **Word Counter**: Real-time count of words processed dynamically from the editor content.

### Preview & Render
- **Rich Markdown Support**: Full GitHub Flavored Markdown (GFM) parsing via `react-markdown` and `remark-gfm`.
- **Code Snippets**: Highlighted code blocks rendered gracefully.

### Tooling
- **Toolbar Utilities**: Quick format insert buttons (Bold, Italic, H1, H2, Link, Code, Divider) utilizing `lucide-react` icons.
- **Export to PDF**: Native functional export wrapping the preview HTML using `html2pdf.js`, generating production-ready print setups with customizable filename.

### UI & Styling
- **Dark Mode**: Fully implemented Light/Dark switch affecting both editor (CodeMirror themes) and the overall UI using Tailwind CSS class-strategy.
- **Modern aesthetics**: Deep Tailwind styling with clean custom scrollbars, cohesive border definitions, and responsive handling.

### Infrastructure
- **Containerization**: Full Docker support out of the box using multi-stage builds.
- **Service Hosting**: Managed via `docker-compose.yml` locally resolving to `http://localhost:3030` backed by an NGINX lightweight server for absolute speed in production.
- **Vite Setup**: Utilizes Vite for blistering fast development cycles.

## 🛠 Tech Stack Snapshot
- **Framework**: React 18, Vite 5
- **Styling**: TailwindCSS 3.4
- **Container**: Docker + Docker Compose, Nginx Alpine image
