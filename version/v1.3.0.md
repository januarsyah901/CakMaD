# 📝 Markdown Editor — v1.3 Content Power-ups

## Overview
Upgrade fokus pada fitur konten yang lebih powerful: table editor, image support,
math rendering, diagram, custom font, dan template starter.

## Prerequisites
- v1.0 sudah jalan (editor, preview, export PDF, dark mode, auto-save)
- v1.1 sudah jalan (resizable panels, scroll sync, keyboard shortcuts, toast)
- v1.2 sudah jalan (multiple tabs, open/save .md, sidebar file explorer)

## Features v1.3

### 1. Table Editor GUI
- Tombol di toolbar buka modal/popover
- User input jumlah baris & kolom via UI
- Generate sintaks markdown tabel otomatis ke editor
- Ada opsi alignment: left, center, right per kolom

### 2. Image Paste / Upload
- Paste gambar dari clipboard langsung ke editor
- Drag & drop file gambar ke panel editor
- Konversi gambar ke base64 dan insert sebagai `![alt](base64...)`
- Batasi ukuran file max 2MB, tampilkan warning jika lebih

### 3. Math Support (KaTeX)
- Render inline math: `$E = mc^2$`
- Render block math: `$$\int_0^\infty$$`
- Tambah plugin `remark-math` + `rehype-katex` ke react-markdown
- Import CSS KaTeX di `main.jsx`

### 4. Mermaid Diagram
- Deteksi code block dengan bahasa `mermaid`
- Render otomatis jadi diagram di panel preview
- Support: flowchart, sequence diagram, gantt, pie chart
- Fallback: tampilkan raw code jika syntax mermaid error

### 5. Font Options
- Dropdown font selector di toolbar atau settings panel
- Font yang tersedia:
  | Font | Source |
  |---|---|
  | Times New Roman | System font (CSS fallback) |
  | Poppins | @fontsource/poppins |
  | Playfair Display | @fontsource/playfair-display |
  | Lora | @fontsource/lora |
  | Fira Code | @fontsource/fira-code |
  | System Default | sans-serif |
- Font hanya berlaku di panel Preview dan hasil Export PDF
- Simpan pilihan font ke localStorage

### 6. Custom CSS untuk Preview
- Tombol "Custom CSS" di settings panel
- Textarea kecil untuk user input CSS override
- Apply langsung ke Preview panel via `<style>` tag dinamis
- Simpan ke localStorage per dokumen

### 7. Template Starter
- Muncul saat buat dokumen baru (modal pilihan template)
- Template yang tersedia:
  | Template | Isi |
  |---|---|
  | Blank | Kosong |
  | README | Header, badges, installation, usage, license |
  | Blog Post | Title, meta, intro, section, conclusion |
  | Resume / CV | Nama, kontak, experience, education, skills |
  | Meeting Notes | Tanggal, attendees, agenda, action items |
  | Technical Doc | Overview, architecture, API reference, examples |
- Template disimpan sebagai konstanta di `src/data/templates.js`

## Build Steps untuk Agent

### Setup
1. Install semua dependencies baru sesuai daftar di atas
2. Import font di `main.jsx`:
```js
   import '@fontsource/poppins'
   import '@fontsource/playfair-display'
   import '@fontsource/fira-code'
   import '@fontsource/lora'
   import 'katex/dist/katex.min.css'
```

### Table Editor
3. Buat `TableEditorModal.jsx` — input rows & cols, generate markdown table string
4. Tambah tombol Table di `Toolbar.jsx`, onClick buka modal
5. Setelah user confirm, insert hasil string ke posisi kursor editor

### Image Support
6. Buat `imageToBase64.js` — fungsi konversi File object ke base64 string
7. Di `Editor.jsx`, tambah event listener `onPaste` dan `onDrop`
8. Jika ada file gambar, konversi lalu insert `![image](base64)` ke editor
9. Tampilkan toast warning jika ukuran file > 2MB

### Math (KaTeX)
10. Install & tambah `remark-math` dan `rehype-katex` ke config react-markdown di `Preview.jsx`
11. Test render inline `$formula$` dan block `$$formula$$`

### Mermaid
12. Buat `MermaidRenderer.jsx` — komponen yang terima string mermaid, render via `mermaid.render()`
13. Di `Preview.jsx`, override component untuk code block: jika `language === 'mermaid'`, render `<MermaidRenderer />`
14. Tambah error boundary supaya diagram yang error tidak crash preview

### Font Options
15. Buat `FontSelector.jsx` — dropdown dengan daftar font
16. Simpan state `selectedFont` di `useMarkdown.js` atau context
17. Apply font ke Preview panel via inline style: `style={{ fontFamily: selectedFont }}`
18. Saat export PDF, inject font-family ke CSS yang dikirim ke html2pdf
19. Persist pilihan font ke localStorage key `md-editor-font`

### Custom CSS
20. Buat `CustomCssPanel.jsx` — slide-in panel atau modal dengan textarea
21. Apply CSS via `<style id="custom-preview-css">` yang di-update tiap perubahan
22. Simpan ke localStorage key `md-editor-custom-css`

### Template Starter
23. Buat `src/data/templates.js` — export array of `{ id, label, content }`
24. Buat `TemplateModal.jsx` — grid card pilihan template
25. Tampilkan modal saat user klik "New Document"
26. Jika user pilih template, set content editor dengan isi template tersebut

### Final
27. Update `exportPdf.js` — pastikan font yang dipilih ikut ter-render di PDF
28. Test semua fitur baru end-to-end
29. Update README dengan dokumentasi fitur v1.3

## Notes untuk Agent
- Mermaid harus di-initialize dengan `mermaid.initialize({ startOnLoad: false })` di useEffect
- KaTeX CSS wajib di-import global supaya formula tidak broken tampilannya
- Font Times New Roman tidak perlu package — cukup `font-family: 'Times New Roman', Times, serif`
- Untuk image drag & drop, pastikan `event.preventDefault()` di `onDragOver` supaya browser tidak buka file
- Custom CSS dari user di-sanitize ringan — minimal cegah `<script>` injection
- Template modal hanya muncul saat dokumen baru & editor masih kosong
